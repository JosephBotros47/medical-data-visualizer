## Project Overview
This project is part of the freeCodeCamp Data Analysis with Python Certification. It visualizes and makes calculations from medical examination data using matplotlib, seaborn, and pandas to explore relationships between cardiac disease, body measurements, blood markers, and lifestyle choices.

## Tasks Completed
- Calculated BMI and added an `overweight` column.
- Normalized cholesterol and glucose data.
- Created a categorical feature count plot split by cardiovascular disease status.
- Cleaned incorrect patient data and generated a correlation matrix heatmap.
